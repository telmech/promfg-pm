Software Policies & Usage Terms

1. Usage Policy

This software (Project & Task Management • Powered by PRO-MFG) is licensed exclusively for internal organizational use. Unauthorized redistribution, reselling, or sub-licensing is strictly prohibited.

2. User Responsibilities

  • Each user is responsible for maintaining the confidentiality of their login credentials.
  • Users must not share accounts or attempt unauthorized access to other accounts.
  • Any misuse, data tampering, or attempts to bypass access controls will result in immediate account deactivation.

3. Data Policy

All project data, task details, comments, and file attachments stored within this system are the property of the organization. The system stores data locally and does not transmit any information to external third-party servers.

4. Privacy Notice

User information (name, email, department, role) is collected solely for authentication and task assignment purposes. No personal data is shared externally. Administrators may view all user records for management purposes as part of their role.

5. File Attachments

Files uploaded as task attachments are stored on the organization's local server under the /uploads directory. It is the administrator's responsibility to ensure appropriate backup and access control for these files.

6. Availability & Support

This software is provided as-is for internal operational use. For technical support, configuration changes, or feature requests, please contact your designated PRO-MFG system administrator.
